Community Board – Fitness Class Schedule
A simple React + Vite project that displays a weekly fitness class schedule using reusable components and data-driven rendering. 
Each class card includes an image, instructor, time, level, and a link for more details.

Features
React components for clean, reusable UI

Class data stored in data.js

Images loaded from the public/ folder

Responsive grid layout

Easy to add or update classes

Project Structure
src/App.jsx – main layout and rendering

src/components/Card.jsx – reusable card component

src/data.js – class information and image paths

public/ – class images

Tech Stack
React, Vite, JavaScript, CSS
